Iniciando tutorial de Python:
- O Python se tornou uma das linguagens de programação mais populares do mundo 
nos últimos anos. E isso se deve, principalmente, à sua versatilidade: o Python 
funciona tanto para o aprendizado de máquinas, construção de sites até para automação 
de tarefas e testes de softwares.

Explicando o que é Python:
- Python é uma linguagem de programação de uso geral. Isso significa que é uma 
linguagem que funciona para criar uma grande variedade de aplicações diferentes. Ou 
seja, o Python é uma linguagem geral, que não é especializada em nenhum problema 
específico. Essa versatilidade e a facilidade de uso para pessoas iniciantes, fez 
com que se tornasse uma das linguagens mais comuns atualmente.

Passo a passo de instalação:
1 - Marque a opção "Add Python to PATH"
2 - Clique em "Install Now"
3 - Se tudo ocorrer bem, a próxima tela será mostrada. Clique em "Close".
4 - Para verificar se a instalação do Python foi bem-sucedida, pesquise no 
menu iniciar por "cmd" e clique duas vezes para abri-lo.


Como configurar o virtualenv:
 - Virtualenv é uma ferramenta que permite criar ambientes Python isolados. Isto é útil para gerenciar dependências e evitar conflitos entre diferentes projetos.

1 - Abra um terminal ou prompt de comando e navegue até o diretório do seu projeto.
2 - Digite o seguinte comando para instalar o virtualenv:
	- pip install virtualenv
3 - Depois que o virtualenv estiver instalado, digite o seguinte comando para criar um novo ambiente virtual:
	- virtualenv venv
4 - Para ativar o ambiente virtual, digite o seguinte comando:
	- source venv/bin/activate (macOS/Linux)
	- venv\Scripts\activate (Windows)
5 - Depois que o ambiente virtual for ativado, você poderá instalar pacotes Python usando pip sem afetar a instalação global do Python.
